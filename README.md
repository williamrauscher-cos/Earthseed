Earthseed Design
A speculative design project by COSIMA Studio.

What this is
Parable of the Sower (Octavia E. Butler, 1993) contains a religion called Earthseed — a set of doctrines, organizing principles, and social structures that its protagonist, Lauren Olamina, develops in a near-future America in collapse. Butler embedded a coherent operating system inside a novel. This project extracts it.
Earthseed.md is the result: a structured Markdown document that translates Butler's world-building into a functional specification for an actual Earthseed community. Not a summary. Not a fan tribute. A document you could hand to a group of strangers and use to begin.

The premise
Butler didn't write a manifesto — she wrote fiction. But the logic inside that fiction is real. Social structures, governance protocols, spatial principles, ritual forms, economic arrangements: they're all there, implied and sometimes explicit, across the two Parable novels.
Our method is analytical fiction — close-reading a novel as if it were a design brief, then producing design artifacts from what we find. The .md is the primary artifact. Everything else in this repo is generated from it.

The genre
This project sits at the intersection of several traditions:
Speculative Design. Design as a method of inquiry, not just production. The .md is a proposal — a thought experiment that happens to be implementable.
Speculative Markdown. The .md file as a creative and analytical form. Fiction interpreted through the lens of information architecture.
Fan fiction as design. Taking a text seriously enough to build from it. The design.md as an act of critical love.
Whole Earth Catalog lineage. Access to tools. The WEC believed that giving people the right information was a form of political action. So does this project.
Open source ethics. The repo format is not incidental. Forking is a political act. Anyone can take this document, adapt it to their context, and start their own chapter. That's the point.

How it works
The .md governs consistency across every instance of itself — but it does not guarantee identical outcomes. Different communities, different places, different people will produce different expressions of the same underlying logic. This is intentional.
The model is Sol LeWitt's wall drawings: instructions written once, executed by many hands, different every time, always recognizably the same work. The instruction is the art. The .md is the instruction.

The .md covers

Doctrine — Core tenets of Earthseed, extracted and paraphrased from Butler's verse
Community Structure — Membership, roles, onboarding, conflict resolution, consensus protocols
Spatial Principles — Site, perimeter, commons, threshold — drawn from the novel's descriptions of Acorn and Bankole Farm
Ritual & Ceremony — Gathering formats, memory practices, naming, initiation, mourning
Economics & Survival — Reciprocity, labour, resource pooling, security logic


Design outputs
The deck (/deck) documents the project concept and includes examples of artifacts that can be generated from the .md:

Site identity systems for Earthseed chapters
Membership and onboarding documents
Community spatial plans
Gathering ritual facilitation decks

These are examples of possible outputs — not canonical implementations. The .md doesn't prescribe a visual language. It provides the structure from which one can be developed, contextually, by whoever is building a chapter.

Using this
To start an Earthseed community: Read Earthseed.md. Use it as a founding document. Fork the repo, adapt the .md to your context, and document what you build.
To contribute: Open an issue or PR. This is a living document. Corrections, additions, and alternative interpretations of Butler's text are welcome, with sourcing.

On fidelity and license
This project is in ongoing dialogue with Butler's text, not a replacement for it. Read the novels. Parable of the Sower (1993) and Parable of the Talents (1998) are the source material and remain their own work.
The design documents in this repo are original work by COSIMA Studio, produced through analytical interpretation of Butler's fiction. They are released for free use and adaptation.
